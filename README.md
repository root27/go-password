# Go-Password

CLI tool to generate random password using Go and Cobra.

---

## Installation

```bash

go install github.com/root27/go-password@latest

```

---

## Homebrew

```bash

brew install root27/tools/go-password

```

or

```bash

brew tap root27/tools

brew install go-password

```

---

## Usage

```bash

go-password -h

```
